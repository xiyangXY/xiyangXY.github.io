安装git
=================

这里介绍的是使用 `yum` 来安装，去服务器执行以下命令
``` cmd
    yum -y install git
```

查询一下是否安装完成

``` cmd
    git --version
```

看到以下结果表明安装完成

![linux-git](/images/linux-git.png)
